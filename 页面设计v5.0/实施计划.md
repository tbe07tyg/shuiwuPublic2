# 🚀 页面设计v5.0 实施计划

## 📋 项目概述

### 项目目标
基于页面设计v5.0的设计文档，实现项目经理AI助手与后台管理系统的完整功能，包括：
- **优先**: 项目经理后台管理系统（6个核心页面）
- **其次**: 项目经理AI助手页面（预留多角色功能）
- **最后**: 团队成员工作台页面
- **集成**: AI助手与后台系统的深度集成

### 技术栈
- **前端框架**: Vue 3 + Composition API
- **UI组件库**: Ant Design Vue
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **图表库**: ECharts
- **AI引擎**: Deepseek API
- **构建工具**: Vite

### 第三步：后台管理系统状态管理

#### 3.1 创建项目管理状态管理
```javascript
// src/store/modules/projectManagement.js
import { defineStore } from 'pinia'

export const useProjectManagementStore = defineStore('projectManagement', {
  state: () => ({
    projects: [],
    currentProject: null,
    projectStats: {
      ongoing: 0,
      completed: 0,
      delayed: 0,
      risky: 0
    },
    loading: false,
    error: null
  }),

  actions: {
    async fetchProjects() {
      this.loading = true
      try {
        // 模拟API调用
        const response = await fetch('/api/projects')
        const data = await response.json()
        this.projects = data.projects
        this.updateProjectStats()
      } catch (error) {
        this.error = error.message
      } finally {
        this.loading = false
      }
    },

    updateProjectStats() {
      this.projectStats = {
        ongoing: this.projects.filter(p => p.status === 'ongoing').length,
        completed: this.projects.filter(p => p.status === 'completed').length,
        delayed: this.projects.filter(p => p.status === 'delayed').length,
        risky: this.projects.filter(p => p.riskLevel === 'high').length
      }
    }
  }
})
```

<script setup>
import { ref, onMounted, nextTick } from 'vue'
import AIAssistantBase from '@/components/ai/AIAssistantBase.vue'
import { message } from 'ant-design-vue'

const inputMessage = ref('')
const loading = ref(false)
const chatContainer = ref(null)
const conversationHistory = ref([])

const projectStats = ref({
  ongoing: 8,
  dueToday: 3,
  risky: 2,
  completionRate: 85
})

const formatTime = (timestamp) => {
  return new Date(timestamp).toLocaleTimeString()
}

const sendMessage = async () => {
  if (!inputMessage.value.trim()) return
  
  try {
    loading.value = true
    // 这里调用AI服务
    const response = '这是AI的回复示例'
    
    conversationHistory.value.push({
      role: 'user',
      content: inputMessage.value,
      timestamp: new Date()
    })
    
    conversationHistory.value.push({
      role: 'assistant',
      content: response,
      timestamp: new Date()
    })
    
    inputMessage.value = ''
    
    await nextTick()
    if (chatContainer.value) {
      chatContainer.value.scrollTop = chatContainer.value.scrollHeight
    }
  } catch (error) {
    message.error('发送消息失败，请重试')
  } finally {
    loading.value = false
  }
}

const generateReport = async () => {
  try {
    loading.value = true
    message.success('报告生成成功')
  } catch (error) {
    message.error('报告生成失败')

    loading.value = false

}
